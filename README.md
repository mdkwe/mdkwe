### My latest project

**[Build or Pass →](https://build-or-pass.vercel.app/)**

You have a startup idea. But is it actually worth building?

Paste your idea and get back a simple breakdown:
- Is this worth exploring?
- What would a first version look like?
- How do you test it quickly?

It's not a business plan. Just a fast gut-check.

Built with React and TypeScript.
