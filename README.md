## What I'm currently working on

**Papaya** — a structured, searchable database of energy data sources, designed to help analysts, builders, and decision-makers find, evaluate, and use the right information faster.

The energy sector runs on fragmented, hard-to-access data — spread across institutions, platforms, reports, and APIs. Papaya is built to bring structure to that complexity.

Instead of just listing resources, Papaya organizes them by:
- energy segment (power, gas, renewables, markets, policy, etc.)
- geography (global, regional, national, market-level)
- data type (raw data, dashboards, reports, APIs)
- use case (analysis, monitoring, investment, research)

Each source is qualified to answer a simple question:  
**when should I use this, and why this one over another?**

Starting with a curated set of high-value sources, Papaya is designed to evolve into a decision-support layer for energy data and intelligence.
